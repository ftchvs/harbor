# Harbor — Distribution

This repository hosts the public distribution artifacts for **Harbor**, a native
macOS menu-bar utility that shows every dev server listening on localhost and
lets you act on it.

- **Download:** [latest release](https://github.com/ftchvs/harbor/releases/latest) (`Harbor.dmg` + SHA256)
- **Landing page:** https://ftchvs.github.io/harbor/
- **Changelog:** [CHANGELOG.md](CHANGELOG.md)
- **Update feed:** `appcast.xml` (Sparkle, EdDSA-signed)

Harbor is a native reimagining inspired by [Ports](https://www.ports-app.com);
it is an independent implementation, not affiliated with or endorsed by the
Ports project.
